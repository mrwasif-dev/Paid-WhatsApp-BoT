## üíª Local Setup

```bash
# Clone repository
git clone https://github.com/mrwasif-dev/Paid-WhatsApp-BoT.git

# Go to directory
cd Paid-WhatsApp-BoT

# Install dependencies
npm install

# Start bot
npm start
```

After starting, open http://localhost:3000 in your browser.
